
1. **ETL (Extract, Transform, Load):**
    
    - Extrai dados de diversas fontes (sistemas, planilhas, web, etc.).
        
    - Transforma e limpa os dados para padronização.
        
    - Carrega em um repositório unificado (Data Warehouse).
        
2. **Data Warehouse:**
    
    - Armazena os dados estruturados de forma organizada.
        
    - Suporta grandes volumes de dados históricos e permite consultas complexas.
        
3. **Ferramentas analíticas:**
    
    - OLAP, mineração de dados, estatísticas, machine learning.
        
    - Análises preditivas e prescritivas.
        
4. **Apresentação de dados:**
    
    - Dashboards, relatórios interativos, gráficos, mapas geográficos.
